# Blinkit-Grocery-Data
This dashboard provides a detailed analysis of retail sales performance across Blinkit outlets. It brings together essential business metrics such as product categories, outlet characteristics, item attributes, and customer sentiment to uncover trends and patterns that support data-driven decision-making.

# Project Overview

The dataset includes the following types of information:

Product categories such as Fruits and Vegetables, Frozen Foods, Health and Hygiene, and Canned items.

Outlet attributes such as outlet type, size, identifier, establishment year, and location tier.

Item-level details such as weight, visibility, rating, and item identifier.

Sales data across multiple outlets.

The dashboard allows users to interact with the dataset and explore how item characteristics and outlet features affect overall sales.
